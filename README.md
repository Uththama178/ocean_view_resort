# Ocean View Resort Management System
## Technologies Used
- Jakarta EE / Servlet
- MySQL / IntelliJ IDEA Ultimate
- Maven
## Setup Instructions
1. Run the `database.sql` script in your MySQL server.
2. Update `DBConnection.java` with your database credentials.
